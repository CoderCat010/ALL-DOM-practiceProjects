Task 1: Button click করলে একটা counter বাড়বে। (0, 1, 2, 3...)

Task 2: Input এ একটা number লিখবে — সেটা জোড় নাকি বিজোড় বলবে

Task 3: একটা button আছে "Show" আরেকটা "Hide" — click করলে একটা text দেখাবে বা লুকাবে

Task 4: Input এ বয়স লিখে button click করলে বলবে — "তুমি প্রাপ্তবয়স্ক" বা "তুমি অপ্রাপ্তবয়স্ক"

Task 5: তিনটা input — তিনটা number। Button click করলে সবচেয়ে বড় number টা দেখাবে।
